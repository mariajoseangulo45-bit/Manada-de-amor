# 🐾 Manada de Amor

**Proyecto educativo — HTML + CSS**

EL Sitio web fue creado para la Fundación Manada de Amor como proyecto educativo, una organización que rescata, cuida y busca hogar para perros en situación de abandono. El objetivo del proyecto fue trasladar esa misión a una página web real: que cualquier persona pueda conocer a los peludos, entender el trabajo de la fundación y ponerse en contacto para adoptar, donar o ser voluntario.

---

## 💡 ¿Qué hace este sitio?

- Presenta la fundación con su historia, misión y visión.
- Muestra un catálogo de perros en adopción, cada uno con su propia página de perfil.
- Incluye un formulario de contacto para adopción, donación, voluntariado e información general.
- Tiene una sección de testimonios con galería de fotos.

## 🔍 Cosas que vale la pena notar en el código

- Cada perrito tiene **su propio par HTML + CSS**, lo que hace fácil editar un perfil sin afectar los demás.
- El formulario usa validaciones nativas de HTML: `required`, `type="email"`, `pattern` para nombres — sin una sola línea de JavaScript.
- La tipografía *Allura* se trae directo de Google Fonts y le da carácter al sitio sin necesidad de archivos externos.
- La navegación del catálogo incluye un botón de regreso en cada perfil, pensando en la experiencia del usuario.
